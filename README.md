
###BingoWorld介绍

	BingoWorld是一款IT阅读学习类的开源软件，在不久的将来会拥有丰富的学习内容，
	包含Android、iOS技术文章和相关面试题等。推荐您也来用用来看看，您也可以发布自己的Bingo链接文章。

	该APP是通过新浪微博登录的，为防止大家clone下来后无法编译和不能正常使用微博登录，现将证书上传，
	如果你还是不能编译，请通过下面的联系方式与我联系。
